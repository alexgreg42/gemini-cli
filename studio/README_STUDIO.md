# Gemini 3 Flash Studio (GUI)

A premium, non-CLI environment for interacting with Gemini 3 Flash.

## Setup

1.  **API Key**: Open the `.env` file in this directory and replace
    `YOUR_API_KEY_HERE` with your actual Gemini API key from
    [Google AI Studio](https://aistudio.google.com/apikey).
2.  **Run**: The studio is already running on
    [http://localhost:5173/](http://localhost:5173/).

## Features

- **Interactive Chat**: Beautiful glassmorphism UI with smooth animations.
- **Gemini 3 Flash**: Powered by the latest Gemini 3 Flash model for fast and
  intelligent responses.
- **Code Highlighting**: Full syntax highlighting for code blocks.
- **Terminal Integration**: Designed to be the "non-CLI" companion to the Gemini
  CLI.

## How to use

Simply type your message in the input area at the bottom and press Enter or
click Send. You can use Shift+Enter for multi-line messages.
